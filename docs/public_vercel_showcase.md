# Public Vercel Showcase

The public showcase is a static Next.js app at `apps/public_showcase`. It has been rebuilt as a Stollery-focused product demonstration rather than an architecture narrative.

Display title: `Reimagining Alberta ED Flow Intelligence`

Primary route: `/Reimagining-Alberta-ED-Flow-Intelligence`

Lowercase redirect: `/reimagining-alberta-ed-flow-intelligence`

## Experience

The public experience has exactly four major sections:

1. Open Data Context
2. Synthetic Stollery ED Operating Reality
3. Blended Predictive Intelligence
4. Scenario & What-If Studio

Snowflake portability, research-to-capability mapping, and standalone lineage/trust sections are intentionally removed from the public surface.

## Data Boundary

The showcase uses Stollery JSON artifacts generated by:

```powershell
python -m ed_flow_kernel.exports.stollery_public_showcase_export --out apps/public_showcase/public/data --seed 20260601
```

Artifacts include schema version, generation timestamp, focus site, data mode, source type, synthetic flag, caveats, and data. They are public-safe and contain no real patient data or secure AHS data.

Baseline forecasts remain fixed in the browser. Scenario controls and presets update comparator outputs through `apps/public_showcase/lib/scenarioEngine.ts`.
