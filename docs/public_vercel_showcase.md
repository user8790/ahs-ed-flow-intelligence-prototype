# Public Vercel Showcase

The public showcase is a static Next.js app at `apps/public_showcase`.

Display title: `Reimagining Alberta ED Flow Intelligence`

Primary route: `/Reimagining-Alberta-ED-Flow-Intelligence`

Lowercase redirect: `/reimagining-alberta-ed-flow-intelligence`

## Data Boundary

The showcase uses JSON artifacts generated by:

```powershell
python -m ed_flow_kernel.exports.public_showcase_export --out apps/public_showcase/public/data --mode public_demo --seed 20260601
```

Artifacts include schema version, generation timestamp, data mode, source categories, lineage, synthetic flag, caveats, and payload. They are public-safe and contain no real patient data or secure AHS data.

## Local Validation

```powershell
cd apps/public_showcase
npm install
npm run typecheck
npm run build
```

## Deployment

Preferred deployment is as a page inside the existing SAO Advisory website:

`https://www.sao-advisory.com/Reimagining-Alberta-ED-Flow-Intelligence`

If that repo is unavailable, deploy the standalone app as `reimagining-alberta-ed-flow-intelligence`. No secrets or environment variables are required.

Current standalone deployment:

`https://reimagining-alberta-ed-flow-intelli.vercel.app/Reimagining-Alberta-ED-Flow-Intelligence`

The SAO domain is currently attached in Vercel to an existing project, so this branch does not overwrite it.
