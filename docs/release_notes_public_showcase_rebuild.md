# Public Showcase Rebuild Release Notes

Updated: 2026-06-01

## Summary

The public Vercel showcase has been rebuilt around a Stollery-only product narrative:

`Reimagining Alberta ED Flow Intelligence`

The old Snowflake/research/lineage-heavy public experience has been replaced with four major sections:

1. Open Data Context
2. Synthetic Stollery ED Operating Reality
3. Blended Predictive Intelligence
4. Scenario & What-If Studio

## What Changed

- Added a Stollery-specific public artifact exporter.
- Generated 14 required `stollery_*.json` artifacts.
- Added public facts and synthetic assumptions documentation.
- Replaced the public Next page with an interactive, responsive, Stollery-focused demo.
- Added deterministic browser-side scenario controls, presets, reset, comparator outputs, bottleneck migration, and huddle brief.
- Added artifact tests for schema, no-PHI checks, forecast shape, unit capacity coherence, scenario validity, and synthetic history range.

## What Was Removed From The Public Story

- Snowflake Portability Story as a major public section
- Research-to-Capability Map as a major public section
- Data Lineage and Trust as a major public section
- Architecture-heavy implementation narrative

## Public Safety Boundary

The public app uses public references, public/open-data-shaped context, and synthetic internal data only. It does not contain real AHS secure data or patient identifiers.
