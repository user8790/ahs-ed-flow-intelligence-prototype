# AHS ED Flow Action Intelligence

Separate Snowflake-portable Streamlit app for operational action interpretation.

Run locally from the repository root:

```powershell
streamlit run apps/streamlit_action_intelligence/app.py
```

This app uses synthetic/local data by default and does not require Snowflake credentials. In Snowflake, deploy this entry point as a new Streamlit app rather than repointing the existing public prototype.

Suggested Streamlit slug: `ahs-ed-flow-action-intelligence`
